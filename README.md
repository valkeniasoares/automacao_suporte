# Automação de Processos Operacionais com Python

## Sobre o Projeto

Este projeto foi desenvolvido para automatizar uma atividade operacional repetitiva realizada diariamente pela equipe de suporte.

A tarefa exigia diversas interações manuais em um sistema web, consumindo tempo significativo da operação e aumentando o risco de falhas humanas decorrentes da repetição constante do processo.

Com o objetivo de otimizar a execução da atividade, foi criada uma automação utilizando Python e PyAutoGUI, capaz de reproduzir as ações necessárias de forma automática, reduzindo o esforço manual e aumentando a produtividade da equipe.

## Tecnologias Utilizadas

* Python
* PyAutoGUI
* Time

## Funcionalidades

* Navegação automática entre telas do sistema
* Execução de cliques e ações repetitivas
* Processamento contínuo de atividades operacionais
* Redução da intervenção manual durante a execução do processo

## Benefícios Obtidos

* Redução do tempo gasto em tarefas repetitivas
* Padronização da execução do processo
* Diminuição da possibilidade de erros operacionais
* Maior disponibilidade da equipe para atividades analíticas e atendimento ao cliente

## Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos relacionados a:

* Automação de processos
* Programação em Python
* Identificação de oportunidades de melhoria operacional
* Resolução de problemas através de tecnologia
* Otimização de rotinas de trabalho

## Observações

Este projeto foi adaptado para fins de demonstração e portfólio, não contendo informações internas, regras de negócio ou dados sensíveis do ambiente corporativo em que a solução original foi utilizada.
